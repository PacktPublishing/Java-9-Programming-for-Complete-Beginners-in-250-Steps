### /src/com/in28minutes/oops/MotorBike.java

```java
package com.in28minutes.oops;

public class MotorBike {

	//state
	private int speed; //member variable
	
	//behaviour
	//method
	//inputs - int speed
	//outputs - void
	//name - setSpeed
	void setSpeed(int speed) { //local variable
		this.speed = speed;
	}
	
	void start() {
		System.out.println("Bike Started");
	}
}
```
